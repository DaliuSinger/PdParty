Description

Run your Pure Data patches on iOS with native GUIs emulated. Inspired by Chris McCormick's PdDroidParty for Android and the (now defunct) RjDj iOS app.

See the PdParty user guide for detailed usage info and how to create and load patches & scenes onto your device:

3rd Party Libraries

This project uses:

  • libpd: audio engine
  • pd externals:
    • ggee: getdir, moog~, stripdir
    • mrpeach: midifile
    • rjlib: rj_accum, rj_barkflux_accum~, rj_centroid~, rj_senergy~, rj_zcr~
  • PGMidi: midi i/o
  • CocoaOSC: Open Sound Control i/o
  • CocoaHTTPServer: WebDAV server
  • ZipArchive: support for decompressing zip archives
  • MBProgressHUD: progress spinner overlay
  • UIAlertView+Blocks
  • UIActionSheet+Blocks
